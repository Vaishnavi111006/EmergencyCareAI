# 🚑 EmergencyCare AI

## Overview

EmergencyCare AI is an AI-powered Heart Disease Risk Prediction System built using Machine Learning. It predicts whether a patient is at high or low risk of heart disease based on health parameters.

## Features

- Heart Disease Prediction
- Confidence Score
- Health Recommendations
- Emergency Contact Numbers
- Nearby Hospital Finder

## Tech Stack

- Python
- Streamlit
- Pandas
- Scikit-Learn
- Joblib

## Dataset

Heart Disease Dataset

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
